\# Ultimate Bot



WhatsApp bot using Baileys.



\## Features



\- Group management: .add, .kick, .promote, .demote, .admins

\- Profile \& info: .getpp, .whois, .jid, .status

\- Bot system: .ping, .alive, .uptime, .menu

\- Fun commands: .quote, .say, .sticker

\- Owner commands: .block, .unblock, .setpp, .setbio
